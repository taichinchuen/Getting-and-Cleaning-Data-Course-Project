# Codebook

This codebook summarizes the R script (run_analysis.R) for this project.

## Instruction

1. Download the file from the website link and unzip the file by download.file() and unzip() function.
2. Read data from the targeted files by read.table() function.
3. Merge the training and the test sets to create one data set by cbind() and rbind() function.
4. Extract only the measurements on the mean and standard deviation for each measurement by grep() and subset() function.
5. Adopt descriptive activity names to name the activities in the data set by factor() function.
6. Label the data set with descriptive variable names by gsub() and names() function.
7. Calculate the average of each variable for each activity and each subject by aggregate() function.
8. Create a second, independent tidy data set by write.table() function.

## Activity Types

* WALKING
* WALKING_UPSTAIRS
* WALKING_DOWNSTAIRS
* SITTING
* STANDING
* LAYING
